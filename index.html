🔍 jlens-gguf - Visualize and steer your machine learning models

Download jlens-gguf

jlens-gguf provides a simple way to observe and influence how GGUF models process information. You can use this software to visualize internal model weights through a Jacobian-Lens approach. This tool allows you to steer the output of your models in real time while they run on llama.cpp.

🚀 Getting Started

You can run this software on your Windows computer without technical knowledge. This guide walks you through the setup process step by step.

💻 System Requirements

  • Windows 10 or Windows 11 operating system.
  • At least 8GB of system memory (RAM).
  • A modern graphics card to handle model processing efficiently.
  • Basic knowledge of how to navigate files in Windows Explorer.

💾 Download and Installation

Follow these steps to acquire and prepare the software on your machine:

  1. Visit the official releases page to download the latest version.
  2. Look for the file ending in .zip or .exe under the most recent version tag.
  3. Download the file to your "Downloads" folder.
  4. If the file is a .zip archive, right-click the file and select "Extract All" to create a new folder.
  5. Open the extracted folder.

⚙️ Running the Visualizer

Once you extract the files, follow these instructions to launch the program:

  1. Locate the file titled jlens-gguf.exe inside the application folder.
  2. Double-click this file to start the application.
  3. Windows may show a security prompt. If this happens, click "More info" and then "Run anyway."
  4. The main application window will appear. It might take a few seconds to load your first model.

🛠 Using the Interface

The interface allows you to connect a GGUF model file and begin the visualization process.

  • Load Model: Use the "File" menu to select a GGUF model file stored on your computer.
  • Jacobian Lens: Toggle this feature to see the internal transformations happening inside the model.
  • Steering: Adjust the sliders in the control panel to change the behavior of the output in real time.
  • Console: The bottom section displays status updates and errors during your session.

💡 Troubleshooting

If you encounter issues while running the program, check the following steps:

  • Ensure your GGUF model file is not corrupted.
  • Close memory-intensive applications if the program becomes unresponsive.
  • Check that your computer meets the minimum requirements listed above.
  • Make sure your graphics drivers are updated to the current version.
